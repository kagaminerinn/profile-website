PM (Project Manager)

ISPM (Information System Project Management) is one of the subjects in semester 5 of the Information Systems department. The objective of this project is that we have to make several elements, such as MoM (Minute of Meeting), blueprint, testing scenario, database design, and also the user manual of an existing application

  • Determine which application to use, here we choose Gojek
  • Do brainstorming ideas and manage project timelines
  • Define project goals, objectives, and scope
  • Divided part of each task to each member
  • Make A wirefrane Design from Gojek App for the blueprint task
  • Develop detailed project plans, including timelines, milestones, and task
